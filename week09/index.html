The Window Object

Supported by all browsers and represents the browser's window. Global Javascript functions, variables and objects automatically become members of the window object. There are also window object methods.

  • One that is commonly used is the alert() method. It displays an alert box with a message and an ok button. Click here to see an example. 

function sendAlert() { 
   alert("This is an alert() method!"); 
}

  • Another method is the scrollTo() method. This allows you to have the page take you to a specified destination on the window. Click here to try the scrollTo() method. Here are two functions to make the scrollTo() methods work: 

                        function scrollBottom() { 
   window.scrollTo(0,5000); 
}
                        function scrollBack() {
  window.scrollTo(0,50); 
}
