The Window Object

Supported by all browsers and represents the browser's window. Global Javascript functions, variables and objects automatically become members of the window object. There are also window object methods.

  • One that is commonly used is the alert() method. It displays an alert box with a message and an ok button. Click here to see an example. 

function sendAlert() { 
   alert("This is an alert() method!"); 
}

  • Another method is the scrollTo() method. This allows you to have the page take you to a specified destination on the window. Click here to try the scrollTo() method. Here are two functions to make the scrollTo() methods work: 

                        function scrollBottom() { 
   window.scrollTo(0,5000); 
}
                        function scrollBack() {
  window.scrollTo(0,50); 
}

Animation Example

Animation with requestAnimationFrame.

                    const squareElement = document.getElementById('square');
let angle = 0;
function rotate() {
   angle = (angle + 2)%360;
   squareElement.style.transform = `rotate(${angle}deg)`
   window.requestAnimationFrame(rotate);
}
const id = requestAnimationFrame(rotate);

The same animation ca be achieved merely by css in the styles.css file using css animation property and @keyframes.

                    @keyframes spin { 
   from { transform:rotate(0deg); } 
    to { transform:rotate(360deg); } 
 }
Canvas practise
Sorry, but your browser does not support the canvas element 
                     const canvasElement = document.getElementById('canvas');

const context = canvasElement.getContext('2d');
context.fillStyle = "#0c0"; // a green fill color
                        
context.strokeStyle = "#ccc"; // a gray stroke color
context.lineWidth = 4;
//  drawing a filled-in rectangle
context.fillRect(10,10,200,100);
// drawing a rectangle that is not filled in
context.strokeRect(10,200,150,70);
                        
                        
/* draw a thick black T shape onto the canvas by moving to the
coordinates (130,50), then drawing a horizontal line 30 pixels long, and finally
moving to the middle of that line and drawing a vertical line 40 pixels long*/
context.beginPath();
context.moveTo(130, 50);
context.lineTo(180, 50);
context.moveTo(155, 50);
context.lineTo(155, 90);
context.strokeStyle = '#000';
context.lineWidth = 15;
context.stroke();


//  drawing a red circle of radius 40 pixels at center (300,300)
context.beginPath();
context.arc(300, 300, 40, 0, Math.PI * 2, false);
context.strokeStyle = '#f00';
context.lineWidth = 4;
context.stroke();

//  draw the text “Hello from Canvas” in blue at coordinates (20,300)
context.fillStyle = '#00c'; // a blue fill color
context.font = 'bold 26px sans-serif';
context.fillText('Hello I am from Canvas', 20, 300);
