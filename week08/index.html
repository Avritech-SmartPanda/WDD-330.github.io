CSS transitions allow you to animate something that may typically happen immediately. You can customize how it is animated and for how long. For example, if you want a button to change color when you hover over it, you can. Below is the code to do that. 


button { 
       width: 80px; 
       height: 80px; 
       color: green; 
       transition: color 2s; 
} 
            
button:hover { 
   color: orange;
}

We first target the button in order to style it and give it a green color. When we target it again but use the :hover, it makes it transition to orange when the user hovers over it. The transition also has a clarification of 2s so that the transition happens over a span of 2 seconds.
